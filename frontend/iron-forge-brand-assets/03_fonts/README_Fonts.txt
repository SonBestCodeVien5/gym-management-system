Iron Forge Gym - Font Setup

Recommended fonts from Google Fonts:
1) Bebas Neue - logo/hero/display text
2) Barlow Condensed - body, nav, labels, buttons

No .ttf/.otf files are bundled in this generated kit. Download the fonts from Google Fonts and install or import them in Figma/Illustrator/Web.

Suggested web import:
<link href="https://fonts.googleapis.com/css2?family=Bebas+Neue&family=Barlow+Condensed:wght@300;400;600;700;900&display=swap" rel="stylesheet">

Fallback used in preview exports: Noto Sans Display Condensed.
