IRON FORGE GYM - Generated Asset Kit

This is a starter asset package based on the provided Brand Identity & Asset Guide.

Contents:
01_logo      - logo variants, favicon, app icon, watermark
02_colors    - palette JSON/CSS/GPL and swatch preview
03_fonts     - font setup README only; no font files bundled
04_social    - social media templates and profile assets
05_print     - starter print-ready PDF/SVG mockups
06_website   - OG image, hero texture, favicon package, icons
07_guidelines - notes and this README
08_mockups   - preview mockups
previews     - contact sheets for review

Important:
- SVG files are the main editable deliverables. Open them in Figma/Illustrator to refine spacing and convert text to outlines.
- Print PDFs are starter layouts and should be preflighted by the print shop before production.
- Fonts are not bundled; use Bebas Neue and Barlow Condensed from Google Fonts.
